My Mission: Music by Humans, for Humans

Music is an essential form of human expression, a craft that connects artists and listeners on a deep, emotional level. I believe this connection is threatened by the rise of mass-produced, AI-generated content.

This project is my stand against the devaluation of music. It's for those who believe that art should have a soul, not just an algorithm. When music is created almost for free, without human passion or effort, only large corporations profit, while real artists struggle to be heard. I aim to preserve the value of human creativity and ensure that music remains a meaningful part of our culture.

Auto Skipper

To use the Auto Skipper, you need to create your own free Spotify App. This is a one-time setup that takes about 2 minutes. Your data is saved only in your browser.

One-Time Setup Guide

  1. Go to the Spotify Developer Dashboard and log in.
  2. Click "Create app". Give it any name and description.
  3. Once created, you will see your Client ID. Copy it.
  4. Click "Edit settings". In the "Redirect URIs" field, add these two exact URLs:
    https://ai-musician-list.tanzstoff.de/
    http://127.0.0.1:8888
  5. Scroll down, check the box for "Web API".
  6. Click "Save" at the bottom of the Spotify form.
  7. Paste your Client ID below and click "Save ID".

Please accept "Personalization" cookies to enable this feature.

Your Client ID is saved in your browser.

Desktop App (Beta)

Run a lightweight script in the background on your PC/Mac. Most reliable.

How to use the Desktop App

The script runs in the background on your PC/Mac and is the most reliable way to use the Auto Skipper.

  1. Install Python: If you don't have it, install Python from python.org. Make sure to check "Add Python to PATH" during installation.
  2. Install Libraries: Open a terminal (Command Prompt or PowerShell on Windows, Terminal on Mac) and run this command:
    pip install spotipy requests
  3. Run the Script: Save the downloaded watcher.py file to a folder. Open a terminal in that folder and run:
    python watcher.py
  4. First Login: A browser window will open for you to log in to Spotify. After you approve, you can close the window.
  5. Autostart (Optional on Windows): To make the script run automatically when your PC starts, press Win + R, type shell:startup, and press Enter. Place a shortcut to the watcher.py file in this folder.

By connecting or downloading, you agree to the use of necessary local storage/files to keep you logged in.

For Mobile Users

Even without live monitoring, you can use this list to improve your listening experience!

  1. Find an artist on the watchlists below.
  2. Click the link to open their profile in your Spotify app.
  3. Tap the three dots (...) on the artist's page.
  4. Select "Don't play this artist".

Spotify will now stop playing music from this artist for you.
